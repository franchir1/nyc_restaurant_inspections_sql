# Q4 – Criticality on Weekends vs Weekdays

## Analysis Question

Are violations of hygiene and food safety regulations **more frequent during weekends**
compared to weekdays?

The working hypothesis is that:
- customer traffic is higher on weekends
- increased operational pressure may lead
  to lower compliance with hygiene standards

---

## Analytical Context

The analysis uses the **star data model**, specifically:

- **Fact table:** `inspection_events_table`
- **Dimensions:**
  - `inspection_dim` (violations and criticality)
  - `date_dim` (weekend / weekday classification)

The observed metric is:
- **violation count**

---

## Analysis Logic

The analysis is structured in two steps:

### 1. Raw analysis
Direct comparison between:
- number of violations occurring on weekends
- number of violations occurring on weekdays

### 2. Normalized analysis
Normalization of the number of violations
by the number of available days:
- **2 weekend days**
- **5 weekdays**

This step is essential to avoid
misleading interpretations due to the different number of days.

---

## SQL Query – Violation Count

```sql
WITH violations_count_list AS (
    SELECT
        COUNT(
            CASE
                WHEN id.violation_code IS NOT NULL THEN 1
            END
        ) AS total_violations,
        COUNT(
            CASE
                WHEN id.violation_code IS NOT NULL
                     AND dd.is_weekend = TRUE
                THEN 1
            END
        ) AS total_weekend_violations
    FROM
        inspection_events_table AS iet
    JOIN
        inspection_dim AS id
        ON iet.inspection_key = id.inspection_key
    JOIN
        date_dim AS dd
        ON iet.date_key = dd.date_key
)
SELECT
    ROUND(
        (CAST(total_weekend_violations AS DECIMAL) / total_violations) * 100,
        2
    ) AS percentage_of_violations_during_weekends
FROM
    violations_count_list;
````

---

## Results – Raw Analysis

| Day type | Number of violations | Percentage |
| -------- | -------------------: | ---------: |
| Weekdays |               74,708 |      71.4% |
| Weekends |               29,903 |      28.6% |

At first glance, violations appear to be more frequent on weekdays.

---

## Normalization by Number of Days

### Rationale

The direct comparison is affected by the fact that:

* weekdays account for **5 out of 7 days**
* weekends account for **2 out of 7 days**

To obtain a fair comparison,
the number of violations is normalized
by the average number of violations per day.

---

## Normalization Calculation (Python)

```python
A = 29903  # weekend violations
B = 74708  # weekday violations

percentage_weekends = round((A / 2) / ((A / 2) + (B / 5)) * 100, 2)
percentage_weekdays = round((B / 5) / ((A / 2) + (B / 5)) * 100, 2)

print(percentage_weekends)   # 50.02 %
print(percentage_weekdays)   # 49.98 %
```

---

## Results – Normalized Analysis

| Day type | Normalized percentage |
| -------- | --------------------: |
| Weekends |                50.02% |
| Weekdays |                49.98% |

---

## Insight

Once the data are normalized:

* **no significant differences emerge**
  between weekends and weekdays
* the average daily probability of a violation
  is essentially the same

This indicates that:

* higher weekend traffic **does not lead**
  to systematically worse hygiene conditions
* management quality depends on structural factors,
  not on the day an inspection occurs
