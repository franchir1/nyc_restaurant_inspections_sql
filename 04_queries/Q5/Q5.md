# Q5 – Establishment Improvement Over Time

## Analysis Question

Do establishments subject to health inspections **show improvement over the long term**
or do they tend to maintain (or worsen) their quality level?

The objective is to measure the **overall effectiveness of corrective actions**
and of the health inspection system by comparing the initial and final inspection scores
for each establishment.

---

## Analytical Context

The analysis uses the **star data model**, specifically:

- **Fact table:** `inspection_events_table`
- **Dimensions:**
  - `establishment_dim` (unique identification of establishments)
  - `date_dim` (temporal ordering of inspections)

The observed metric is:
- `score_assigned` (inspection score)

> Note: a **lower score indicates better conditions**, while a **higher score indicates greater criticality**.

---

## Analysis Logic

For each establishment:

1. identify the **first inspection** available in the dataset
2. identify the **last inspection** available
3. compare the two scores:
   - `last_score < first_score` → **improvement**
   - `last_score ≥ first_score` → **no improvement**

The analysis therefore provides a **longitudinal view**
of establishment behavior over time, regardless of the number of inspections performed.

---

## SQL Query

The query uses:

- **CTEs**
- **window functions** (`FIRST_VALUE`)
- final aggregations

```sql
WITH scores_list AS (
    SELECT DISTINCT
        establishment_key,
        FIRST_VALUE(score_assigned) OVER (
            PARTITION BY establishment_key
            ORDER BY date_key ASC
        ) AS first_score,
        FIRST_VALUE(score_assigned) OVER (
            PARTITION BY establishment_key
            ORDER BY date_key DESC
        ) AS last_score
    FROM
        inspection_events_table
),
final_counts AS (
    SELECT
        COUNT(DISTINCT establishment_key) AS total_establishments,
        COUNT(
            CASE
                WHEN last_score < first_score THEN 1
            END
        ) AS improved_establishments
    FROM
        scores_list
)
SELECT
    ROUND(
        (CAST(improved_establishments AS DECIMAL) / total_establishments) * 100,
        2
    ) AS improved_percentage
FROM
    final_counts;
````

---

## Output

The final result was exported as a CSV file:

* `improved_%.csv`

---

## Key Results

|     Establishment category     |        SQL criterion       | Percentage |
| :----------------------------: | :------------------------: | :--------: |
|   🟢 Improved establishments   | `last_score < first_score` |   **22%**  |
| 🔴 Not improved establishments | `last_score ≥ first_score` |   **78%**  |

* Approximately **22%** of establishments show a **net improvement**
  between the first and last inspection
* The remaining **78%**:

  * maintain a similar level
  * or show a deterioration

---

## Insight

The analysis highlights that:

* the inspection system **does produce improvements**, but only for a minority
* many establishments:

  * temporarily address critical issues
  * but do not show a **structural long-term improvement**

This suggests that:

* corrective actions are often effective **in the short term**
* but do not always lead to lasting changes
  in internal operating processes
