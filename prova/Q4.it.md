# Q4 – Criticità nei giorni festivi vs giorni feriali

## Domanda di analisi

Le violazioni delle norme igienico-sanitarie sono **più frequenti durante il fine settimana**
rispetto ai giorni feriali?

L’ipotesi di partenza è che:
- nei weekend l’afflusso di clienti sia maggiore
- una maggiore pressione operativa possa portare
  a una minore attenzione al rispetto delle norme igieniche

## Contesto analitico

L’analisi utilizza il **modello dati a stella**, in particolare:

- **Fact table:** `inspection_events_table`
- **Dimensioni:**
  - `inspection_dim` (violazioni e criticità)
  - `date_dim` (classificazione weekend / feriale)

La metrica osservata è:
- **conteggio delle violazioni**

## Logica dell’analisi

L’analisi è articolata in due fasi:

### 1. Analisi grezza
Confronto diretto tra:
- numero di violazioni avvenute nei weekend
- numero di violazioni avvenute nei giorni feriali

### 2. Analisi normalizzata
Normalizzazione del numero di violazioni
rispetto al numero di giorni disponibili:
- **2 giorni** di weekend
- **5 giorni** feriali

Questo passaggio è fondamentale per evitare
interpretazioni distorte dovute alla diversa numerosità dei giorni.


## Risultati – Analisi grezza

| Tipo di giorno | Numero violazioni | Percentuale |
| -- | -: | -: |
| Giorni feriali |            74 708 |       71.4% |
| Weekend        |            29 903 |       28.6% |

A prima vista, le violazioni sembrano più frequenti nei giorni feriali.

## Normalizzazione per numero di giorni

### Motivazione

Il confronto diretto è influenzato dal fatto che:

* i giorni feriali sono **5 su 7**
* i weekend sono **2 su 7**

Per ottenere un confronto corretto,
si normalizza il numero di violazioni
sul numero medio di violazioni per giorno.

## Calcolo della normalizzazione (Python)

```python
A = 29903  # violazioni nel weekend
B = 74708  # violazioni nei giorni feriali

percentage_weekends = round((A / 2) / ((A / 2) + (B / 5)) * 100, 2)
percentage_weekdays = round((B / 5) / ((A / 2) + (B / 5)) * 100, 2)

print(percentage_weekends)   # 50.02 %
print(percentage_weekdays)   # 49.98 %
```

## Risultati – Analisi normalizzata

| Tipo di giorno | Percentuale normalizzata |
| -- | --: |
| Weekend        |                   50.02% |
| Giorni feriali |                   49.98% |


* **non emergono differenze significative**
  tra weekend e giorni feriali
* la probabilità media di violazione per giorno
  è sostanzialmente identica

Questo indica che:

* l’aumento dell’afflusso nel weekend **non comporta**
  un peggioramento sistematico delle condizioni igieniche
* la qualità della gestione dipende da fattori strutturali,
  non dal giorno in cui avviene l’ispezione
